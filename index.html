HTML Table Reference

Basics
Tag Name Description
<html> ... </html> HTML Declares the Web page to be written in HTML
<head> ... </head> Head Delimits the page's head
<title> ... </title> Title Defines the title (not displayed on the page)
<body> ... </body> Body Delimits the page's body
<h n> ... </hn> Header n Set ... in boldface
<b> ... </b> Bold Set ... in boldface
<i> ... </i> Italic Set ... in italics
<center> ... </center> Center Center ... on the page horizontally
<ul> ... </ul> Unordered List Brackets an unordered (bulleted) list
<ol> ... </ol> Ordered List Brackets a numbered list
<li> ... </li> List Item Brackets the list item
<br> Break Force a line break here
<p> ... </p> Paragraph Brackets a paragraph
<hr> Horizontal Rule Inserts a horizontal rule
<img src="..."> Image Display an image here
<a href="..."> ... </a> Anchor Delines a hyperlink


Table
Tag Name Description
<table> ... </table> Table The wrapper element for all HTML tables.
<thead> ... </thead> Table Head The set of rows defining the column headers in a table.
<tbody> ... </tbody> Table Body The set of rows containing actual table data.
<tr> ... </tr> Table Row The table row container.
<td> ... </td> Table Data The data cell container
<th> ... </th> Table Header the Cells defining the column headers in a table.
<tfoot> ... </tfoot> Table Foot The set of rows defining the footer in a table.
Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
scope Scope Defines what this header's scope is (for accessability)
